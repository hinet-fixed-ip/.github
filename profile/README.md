## 場景一：家裡光世代用戶，想申請固定 IP

這是最原始的需求。中華電信的光世代預設是動態 IP，每次重新撥接 IP 可能就換了。如果你有以下用途，你會需要 hinet 固定 ip：

- 在家架設 NAS，要從外部連進來
- 自建監視系統，需要固定位置讓手機 App 連線
- 架設小型個人網站或測試伺服器
- 搭 VPN 回家，需要固定入口

中華電信的固定 IP 方案叫做「靜態 IP」加購方案，通常要另外跟電信申請，費用依方案不同。設定方式是在原本的帳號後面加上 `@ip`，透過 PPPoE 撥接直接拿到靜態 IP，不需要動小烏龜（家用路由器）的設定。

這個方式適合真的在台灣本地用光世代的人。但如果你不是在台灣本地、或者需要的是一個「像台灣本地用戶」的 IP 環境，那就要往下看了。
